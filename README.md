# 🏠 Hostel Budget Planner

A simple Python calculator that helps students estimate their monthly hostel expenses before moving to college.

## 🤔 Why I built this

I'm a Class 12 student applying to colleges. I wanted to know how much pocket money I'd actually need in college. My older cousin told me expenses add up fast, so I built this to get a realistic estimate.

## ✨ Features

- Calculates fixed expenses (mess, rent)
- Tracks variable expenses (travel, laundry, stationery)
- Includes "fun money" category (snacks, eating out, random stuff)
- Gives personalized saving tips based on spending patterns
- Saves your budget to a text file
